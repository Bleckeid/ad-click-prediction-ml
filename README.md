# Ad Click Prediction 

## Описание проекта
Проект посвящен разработке модели машинного обучения для предсказания вероятности клика пользователя по рекламному объявлению (Click-Through Rate). Модель позволяет оптимизировать маркетинговые бюджеты за счет более точного таргетинга.

## Cтек
* **Python 3.9+**
* **Scikit-learn**: Pipeline, ColumnTransformer, LogisticRegression, SVC.
* **Feature Engineering**: Target Encoding (для высококардинальных признаков), StandardScaler.
* **Калибровка**: Isotonic Regression (CalibratedClassifierCV).
* **Метрики**: PR-AUC, Brier Score, ECE (Expected Calibration Error).

## Решения
1. **Борьба с дисбалансом**: Использование весов классов и метрики Precision-Recall AUC (0.38).
2. **Профессиональная калибровка**: Снижение ошибки калибровки (ECE) с 0.27 до **0.009**, что позволяет использовать предсказания как реальные вероятности.
3. **MLOps**: Реализован скрипт сохранения и проверки работоспособности модели (артефакты: препроцессор, модель, список признаков).

## Структура 
- `dataset/` — исходные данные (не включены в репозиторий).
- `model/` — сохраненные артефакты обученной модели.
- `notebook.ipynb` — основной файл с полным циклом EDA и обучения.
- `requirements.txt` — список зависимостей.

## Результаты
* **PR-AUC**: 0.379
* **Brier Score**: 0.127
* **ECE**: 0.009
* Основные факторы влияния: `site_domain`, `device_model`.
